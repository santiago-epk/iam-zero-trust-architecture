# Identity and Access Management (AAA) & Zero Trust

## Overview
This repository focuses on Authentication, Authorization, and Accounting (AAA) controls within enterprise and cloud-native environments, emphasizing the principle of least privilege and Zero Trust Architectures (ZTA)[cite: 6, 7].

## Core Research & Implementations
* **Corporate Access Control Policy:** A formal IAM policy aligned with NIST CSF (PR.AC-1, PR.AC-4) designed to mitigate brute-force attacks, phishing, and insider threats in remote-work environments[cite: 6].
* **Multi-Factor Authentication:** Research and deployment strategies for One-Time Password (OTP) software implementations[cite: 7].
* **Zero Trust Architecture:** Analysis of NIST SP 800-207A principles, focusing on granular access controls and identity-based policies in multi-location cloud environments[cite: 7].
